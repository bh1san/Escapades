# Preview Project Workflow

1. **Start the Next.js development server**:
   Run `npm run dev` to start the preview server on localhost (by default on port 9002 in this project).

   // turbo
   ```bash
   npm run dev
   ```

2. **Access the application**:
   Once the server starts, open a browser and navigate to `http://localhost:9002` to preview the project.
