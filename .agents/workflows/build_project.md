# Build Project Workflow

1. **Install dependencies**:
   Run `npm install` to install all necessary packages from `package.json`.

   // turbo
   ```bash
   npm install
   ```

2. **Build the project**:
   Run `npm run build` to compile and build the Next.js application for production.

   // turbo
   ```bash
   npm run build
   ```
